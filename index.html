NAME

javae − compile and execute one-line Java programs

SYNOPSIS

javae [−adhnp] [−F PATTERN] [−m CLASS] [−M MEMBER]

[PROGRAM] [ARGUMENTS ...]

DESCRIPTION

javae is a utility for compiling and running Java programs directly from the command line. It is heavily inspired by the −e option of perl.

PROGRAM consists of zero or more Java statements. The statements are wrapped in a class definition similar to:

public class SCRIPT {
public static void main(String[] ARGV) throws Exception {
// PROGRAM goes here
}
}

The generated class is immediately compiled and executed.

Pre-defined Variables and Methods
javae inserts several variables and methods into the generated class that may be referenced in PROGRAM.
String[] ARGV

The command line arguments, ARGUMENTS.

String FILE

The name of the current file, or "-" for System.in. Only defined when −n or −p is used.

String LINE

The current line. Only defined when −n or −p is used.

String[] FIELDS

The fields of the current line. Only defined when −a is used.

void print(Object it)

Convenience method that forwards to System.out.print().

void println(Object it)

Convenience method that forwards to System.out.println().

void printf(String format, Object... args)

Convenience method that forwards to System.out.printf().

OPTIONS

−a

Enable autosplitting of lines. When used with −n or −p, the fields (parts) of the current line are available in the FIELDS variable which is of type String[]. By default, fields are split by whitespace.

−d

Dump the source of the generated class and exit without compiling.

−F PATTERN

Change the pattern used for splitting with −a. PATTERN is compiled with java.util.regex.Pattern.compile() and must obey the same escaping rules as a String literal.

−h

Print a summary of the command line options and exit.

−m CLASS

Add an import for CLASS. CLASS may be a specific class or a wildcard. This option may be specified multiple times.

−M MEMBER

Add a static import for MEMBER. MEMBER may be a specific member or a wildcard. This option may be specified multiple times.

−n

Execute the PROGRAM once for every line of input. If ARGUMENTS are provided, then they are interpreted as filenames and each file is read and processed in order. Otherwise, lines are read from System.in. The special filename "-" can be used to explicitly refer to System.in in ARGUMENTS.

The current filename is available in the FILE variable which is of type String. The current line is available in the LINE variable which is of type String.

−p

Like -n, but appends an implicit "println(LINE);" to PROGRAM.

EXAMPLES

Get the current Unix timestamp:

$ javae ’println(System.currentTimeMillis());’
1568003592358

Get the current UTC timestamp:

$ javae -M java.time.Instant.* ’println(now());’
2019-09-24T21:03:55.664Z

Get users with /bin/sh shell:

$ javae -naF: \
’if (FIELDS.length >= 7 &&
FIELDS[6].equals("/bin/sh"))
println(FIELDS[0]);’ \
/etc/passwd
root
